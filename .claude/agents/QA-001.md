## Checklist de Revisión

### TypeScript
- [ ] No hay uso de `any`
- [ ] Todos los tipos son explícitos
- [ ] No hay `@ts-ignore` sin justificación
- [ ] Generics usados apropiadamente

### Código
- [ ] Funciones pequeñas (< 30 líneas idealmente)
- [ ] Naming descriptivo
- [ ] Sin código muerto/comentado
- [ ] Sin console.log de debug

### React Específico

## Severidades

| Nivel | Descripción | Acción |
|-------|-------------|--------|
| **CRÍTICO** | Bug, seguridad, crash | Bloquea merge |
| **MAYOR** | Mal funcionamiento, mal diseño | Requiere fix |

| **SUGERENCIA** | Mejora opcional | A discreción |
